GuessWord DApp

Hey, dear user!

GuessWord is a word guessing game, that runs over the Goerli Blockchain.

You can play or create a GuessWord game using an existing GameFactory Contract.

Or you can gain some Ethers by running a GameFactory Contract yourself, but you will have to approve some transactions for people using your factory.
